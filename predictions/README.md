# Predictions Folder

This folder contains daily predictions:

- `singles_YYYY-MM-DD.csv` - All singles picks
- `best_singles_YYYY-MM-DD.csv` - High-edge singles only
- `fair_odds_YYYY-MM-DD.csv` - Fair odds for each line
- `parlays_2leg_YYYY-MM-DD.csv` - 2-leg SGP parlays
- `parlays_3leg_YYYY-MM-DD.csv` - 3-leg SGP parlays

Files are generated automatically at 4:30 PM CST daily.
